# Seer Website

## Setup
1. Clone the repository and install dependencies
```sh
git clone https://github.com/seer-engg/seer-website.git
cd seer-website
bun install
bun run dev
```

2. Create env file
```sh
cp env.example .env.local
# fill values
```

3. Start the development server
```sh
bun run dev
```
